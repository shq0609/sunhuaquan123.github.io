hexo搭建个人博客

1.环境搭建

安装nodejs

ndoe -v npm -v 测试是否安装成功

安装hexo

npm install -g hexo-cli （-g表示全局安装hexo-cli脚手架）

验证是否安装成功 hexo -v

切换npm国内源
npm i nrm -g
nrm ls
nrm use taobao

安装git

git –version

git在安装后需要配置一下用户名和邮箱信息，在命令行工具中输入如下命令：
git config –global user.name “你的用户名”
git config –global user.email “你的邮箱”

gitm公钥配置见git文档

2.使用Hexo开始搭建博客

hexo init 项目目录下初始化
hexo n “文件名” 新建文件
hexo clean 清楚
hexo g 重新解析
hexo s 启动服务

3.将博客部署到github

登陆github创建仓库

创建仓库名“账户名.github.io”

在本地目录中安装github插件

hexo项目目录下输入cmd
npm install -save hexo-deployer-git （save表示仅在本项目中安装）

修改hexo项目目录下配置文件_config.yml

_config.yml下修改
deploy:
type: ‘git’
repo: github仓库地址
branch: master

git需要配置公钥

4.更换Hexo博客默认主题

下载主题

从github clone主题文件到hexo项目theme目录下
git clone https：//xxx.git themes/文件夹名称

修改_config.yml

修改该文件下themes位更换的文件夹名称

更新并重新部署

hexo clean 清理缓存
hexo g 解析生成
hexo s 运行hexo服务预览博客
hexo d 部署github通过github链接
