linux基本操作

linux查看软件安装情况
查看软件是否安装
rpm -qa|grep mysql
查看软件包安装的文件
rpm -ql mysql-community-libs-5.6.50-2.el7.x86_64
查看mysql所有安装包的文件存储位置
rpm -qal |grep mysql
yum search 查找对应可以安装的软件包
find / -name mysql 有包含mysql服务的文件路径
Which查找命令which mysql
Whereis命令whereis mysql find类似
