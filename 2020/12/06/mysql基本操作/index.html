mysql基本操作

配置mysql开机自启动
1.chkconfig –add mysql
2.chkconfig mysqld on
命令启动/关闭mysql实例
1.service mysqld start/stop
2./etc/init.d/mysqld start/stop
命令关闭MySQL
mysqladmin -p -u root shutdown
检查mysql是否真正的启动
netstat -tulpn | grep 3306
查询进程
ps -ef | grep mysqld

增加删改查
https://my.oschina.net/u/4115134/blog/3189198

连接mysql服务器
mysql -uroot -proot -u用户名 -proot
连接mysql服务器并指定IP和端口
mysql -uroot -proot -h127.0.0.1 -P3306 -h IP服务器主机名IP地址 3306位默认端口
注释 #、/**/、–添加注释

数据库及表操作
1.创建删除查看数据库
查看数据库
show databases;
进入某数据库进行操作
use 库名；
查看已进入的库
SELECT DATABASE（）；
查看当前数据库中的所有表
show tables;
删除数据库
drop database mydb1;
drop database if exists mydb1;
创建mydb1库，指定编码为utf8
create database mydb1 charset utf8;
create database if not exists mydb1 charset utf8;
查看建库时的语句（并验证数据库库使用的编码）
show create database mydb1;
2.创建、删除、查看表
进入mydb1库，删除stu学生表(如果原本存在)
use mydb1;
drop table if exists stu;
创建stu学生表

查看STU学生表结构
desc stu；
