HTML和CSS基础

网页组成部分：结构、表现、动作。
结构：内容指的是我们页面中看到的数据，通过HTML技术实现。
表现：指的是内容页面上的展示形式。通过CSS技术实现。
动作指的是页面中元素与输入设备交互的响应，用JavaScript实现。

0 HTML基础

超文本标记语言 HTML（Hyper Text Mark Language）
HTML通过在文本文件（网页）中添加标记符，告知浏览器如何显示内容。

  1. 标签格式

0 CSS基础

CSS层叠样式表单，用于控制网页样式并允许样式信息与网页内容分离的标记语言。
选择器{属性：值；}
CSS 注释：/注释内容/

1 CSS和HTML结合方式

  1. 在标签的style属性上设置”key:value value;” 修改标签样式。
  2. 第二种在head标签中，使用style标签来定义各种需要的CSS样式。
  1. 把css样式写成一个单独的css文件，再通过link标签引入即可复用。
